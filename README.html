代码规范

1. 单个组件代码量不允许超过 500 行
2. 重复代码不允许出现三次及以上
3. 数据尽可能放在根目录或者父级目录，并确保 props 单向数据流

项目安装

配置npm

npm install

安装+配置yarn

npm install -g yarn
yarn install

检查yarn版本

yarn --version

Yarn设置淘宝镜像

查询yarn镜像

yarn config get registry

设置成淘宝镜像

yarn config set registry https://registry.npm.taobao.org/        (旧地址)
yarn config set registry https://registry.npmmirror.com/         (最新地址)

清除缓存

yarn cache clean

本地环境(本地开发配置文件.env.dev | .env.dev.local)

yarn dev

生产环境(打包到生产环境配置文件 .evn.prod)

yarn build

安装 第三方组件库（以lodash为例）

npm install lodash --save   # npm
yarn add lodash             # Yarn
npm install lodash -g  	 # 全局 npm
yarn global add lodash   # 全局 Yarn

TypeScript 第三方库有@types类型文件

npm install --save-dev @types/lodash
yarn add --dev @types/lodash

TypeScript 没有类型文件

vue2与vue3通信

src\hooks\useMessage.ts
